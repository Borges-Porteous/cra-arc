Before you start

To use Auto-fill my return (AFR), you must first use the current year’s version of the UFile CVITP software to submit the authorization request, then follow the step-by-step instructions to download the individual’s information.

Remember the following:

  • Close all web browsers that you have running before you attempt to download an individual’s information through AFR
  • Have your CRA user ID and password or sign-in partner information available. This is the same information you would use to access other CRA login services, such as Represent a Client, My Account, and My Business Account. You will need these later in the process
  • Set the time and time zone correctly on your computer

Type of information you wish to provide about your spouse

Text version

UFile screen
Under Interview tab
Add spouse sub-tab is highlighted
Type of information you wish to provide about your spouse page

You must provide spousal information before you submit an Authorization request for the Family Head:

  • Click add spouse
  • Complete the Identification and CRA questions sections for the spouse
  • Return to the Family Head’s profile in the navigation bar and proceed with the AFR process

If you are also using AFR to prepare the spouse’s tax return, you must submit a separate Authorization request for the spouse. This can be done at any time after completing the Identification and CRA questions for the spouse.

Instructions for in person settings

  • Under the Tax Return tab, click CRA Authorization request in the left-side menu to add the Authorization request – signature page to the left-side menu
  • Print the Authorization request – signature page
  • Read the script to the individual
  • Have the individual sign the page

Only the individual keeps a copy of this page. As a volunteers, you do not retain paper copies.

 Instructions for virtual settings

  • Under the Tax Return tab, click CRA Authorization request in the left-side menu to add the Authorization request – signature page to the left-side menu
  • Read the script for virtual situations to the individual
  • Submit the authorization request once verbal permission has been received

Tax information web services

Text version of this screenshot

Tax information web service screen
Sign-In Partner Login/Register button is highlighted
CRA login button is highlighted

Your web browser launches the CRA's Tax information web service page.

Use one of the two methods to access the AFR application:

  1. If you are using the CRA login method, enter your CRA user ID and password
  2. If you are using the Sign-In Partner Login method, log in using your online banking information

This is the same information you would use to access other CRA login services, such as Represent a Client, My Account, and My Business Account. If you run into issues with Multi-factor authentication, refer to Multi-factor authentication – Auto-fill my return (AFR) and CVITP volunteers.

Important note

Although the authorization request is transmitted through the current year tax software, you must still answer Yes to this question in 2017 UFile.

  • In the left side menu, click EFILE
  • Select Yes from the drop-down menu asking if you will be filing an authorization request to use the CRA Auto-fill my return service
  • Click Next

This brings you to the T4 and employment income page.

Important note

Form T1013 was replaced by the Authorization request – signature page beginning in the 2019 tax season. Any reference to Form T1013, Authorizing or Cancelling a Representative in the 2017 software is the same as a reference to the Authorization request – signature page in 2019 and subsequent versions of the software.
